### Spotify data source override

Set `VITE_SPOTIFY_DATA_URL` to a full JSON endpoint URL if you want the Spotify widget to read from a different data source (for example local testing or a staging feed).  
If not set, the app uses `https://raw.githubusercontent.com/jfladas/portfolio/main/public/spotify.json`.
